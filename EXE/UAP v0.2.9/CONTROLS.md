# UAP : Unité Anti-Parasite

## CONTROLS

### QWERTY Keyboards
- Go left			=> A key or Left arrow
- Go right			=> D key or Right arrow
- Pause				=> Space bar
#### [Unreleased]
- Shoot laser		=> W key or Up arrow

### AZERTY Keyboards
- Go left			=> Q key or Left arrow
- Go right			=> D key or Right arrow
- Pause				=> Space bar
#### [Unreleased]
- Shoot laser		=> Z key or Up arrow
